# .vim My vim dot file
Backup your ~/.vimrc
```
cp ~/.vimrc ~/.vimrcbackup
```
```
rm ~/.vimrc
```
cd ~/
clone this profile to you ~/.vim folder.
```
git clone https://github.com/iyuangang/.vim.git
```
open vim
```
vim
```
Input
```
:PlugInstall
```
Enjoy your fast and beauty new vim.

# Remarks

```
:PlugUpdate
```
