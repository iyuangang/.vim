Backup your ~/.vimrc
```
cp ~/.vimrc ~/.vimrcbackup
```
```
rm ~/.vimrc
```

clone this profile to you ~/.vim folder.
```
cd ~/ ; git clone https://github.com/iyuangang/.vim.git
```

open vim
```
vim
```
Input
```
:PlugInstall
```
Enjoy your fast and beauty new vim.

# Remarks
Check update 
```
:PlugUpdate
```
